• Links
    • This is [an example](http://example.com/ "Title") Inline link
    • See my [About](/about/) page for details
    • Automatic Links: <http://example.com/>
  • Image
    • ![Alt text](/path/to/img.jpg)
    • ![Alt text](/path/to/img.jpg "Optional title")
  • Paragraphs --- simply one or more consecutive lines of text, separated by one or more blank lines
  • Headings
    • # This is an H1
    • ## This is an H2
    • ###### This is an H6
  • Preformatted text
    • bold --/**text--/**
    • italic -/*text-/*
  • Lists
    • Use asterisks (*), pluses (+), and hyphens (-) as list markers:

      * itemA
      * itemB
      * itemC

    • Ordered lists use numbers followed by periods:

      1. first item
      2. second item
      3. third item

  • Code Blocks: --- indent every line of the block by at least 4 spaces or 1 tab
    and for syntax highlighting add {:class=language} on next line
